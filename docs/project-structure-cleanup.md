# Project Structure Cleanup

最終更新: 2026-02-17

## 今回実施した整理

- スナップショット保存先を `JS/XML` から `XML` へ移設（実装側も追従）
- 参照されていないバックアップ系ファイルを `archive/backup-files` に集約
- コピー系データを `archive/copied-files` に集約

## 現在の整理ルール

- 実行時に読み書きされるデータは用途別トップレベルへ配置する
  - 例: XML/JSONスナップショットは `XML`
- 退避・履歴用ファイル（`.bak`, `- コピー` など）は `archive` 配下に集約する
- 参照される可能性があるファイルは、移動前に全文検索で参照有無を確認する
